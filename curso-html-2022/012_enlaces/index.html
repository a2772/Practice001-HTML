Enlaces o links

Enlaces

Son los links que conectan una página web con un recurso interno u externo.

Tienen el tributo obligatorio href donde se especifica la ruta del recurso o sitio

También cuenta con el atributo target el cual configura la manera en la que queremos visualizar dicho recurso o sitio.

Tipos de rutas

Absolutas

Usan un protocolo http o https, son únicas en la red y se SUELEN usar para rutas externas

Relativas

No usan protocolo, se basan en la ruta donde se ubica el archivo en el cual se pone la ruta, podemos regresar o movernos entre carpetas
