Tipos de elementos en un formulario (parte 2)

Teoría

Etiquetas

fieldset
Agrupa elementos de un formulario
legend
Se usa para representar con una etiqueta el contenido del fieldset
file
Se usa para cargar archivos y enviarlos desde un formulario
meter
Representa un valor dentro de un rango conocido
progress
Representa el progreso de una tarea
textarea
Se utiliza para introducir texto en un fomrulario

Atributos

placeholder
Da una pista al usuario de lo que se debe introducir
Required
Hace que un campo sea obligatorio de introducir
readonly
Hace que el campo sea de solo lectura
min-max
Establecen el valor mínimo o máximo de un campo numérico
minlenght-maxlenght
Establece el mínimo y máximo de caracteres en un campo de texto
selected
Sirve para marcar como seleccionada una opción
disabled
Desactiva un campo, no se podrá escribir en el autofocus
