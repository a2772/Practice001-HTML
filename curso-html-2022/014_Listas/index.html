Listas ul, ol y dl

Las listas HTML sirven para listar contenido En función del tipo de contenido de nuestra lista tenemos esos 3 tipos de listas.

Unordered list

Se utilizan cundo el orden de los elementos no influye

  • Zanahorias
  • Papas
  • Limones

Ordered list (ol)

Se utilizan cuando el orden de los elementos es importante

  1. Medalla oro
  2. Medalla plata
  3. Medalla brondze

Definition list (dl)

Se usan para hacer un lista de definiciones

Definition term
Definition description
Definition term 2
Definition description 2
