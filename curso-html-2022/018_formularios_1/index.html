Tipos de elementos en un formulario

Teoría

Elementos básicos de un formulario

form
Es la etiqueta que engloba el formulario.
label
Se usa para escribir el nombre del campo a rellenar.
input
Sirve para crear un campo para que el usuario pueda interactuar, usualmente le pedimos que seleccione o llene el input con determinada información
button
Es el botón que nos permitirá enviar el formulario.

Tipos de input

La etiqueta input puede tener varios tipos de entrada de dato según le indiquemos.

  • button: Visualmente igual que un botón pero no envía formularios
  • submit: Se usa para enviar un formulario
  • date: Se usa para introducir fechas
  • date-time: Se usa para fecha y hora pero no funciona en firefox
  • time: se usa para introducir
  • month: Se usa para introducir un mes
  • week: Se usa para introducir el número de semana del año
  • color: Permite seleccionar un color
  • email: Se usa para introducir un email
  • number: Solo admite números
  • password: se usa para introducir contraseñas
  • range: Establece un rango
  • reset: Reinicia el formulario
  • search: Se usa para las barras de búsqueda
  • tel: Se usa para introducir números telef´nicos
  • text: Valor por defecto
  • url: Se usa para introducir urls

Input como seleccionable

  • radio: Permite seleccionar elementos de una lista de items
  • checkbox: Permite marcar o desmarcar un valor

Seleccionables (desplegables principalmente)

  • radio: Permite seleccionar elementos de una lista de items
  • checkbox: Permite marcar o desmarcar un valor
